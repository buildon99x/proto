# AI 프로토타입 공장 Brief

## 목표

한 문장 아이디어를 입력하면 실행 가능한 마이크로 프로토타입, 기획 문서, 프로젝트 메타데이터, 브라우저 검증 증거를 한 번에 생성한다.

## 사용자

- 아이디어를 빠르게 플레이 가능한 형태로 확인하려는 기획자와 개발자
- Hermes 또는 ChatGPT로 만든 아이디어를 반복 가능한 레포 워크플로에 연결하려는 사용자

## 핵심 경험

1. 사용자가 영문 프로젝트 이름과 자유 형식 아이디어를 입력한다.
2. 공장이 AI 또는 결정론적 오프라인 모드로 안전한 구조화 블루프린트를 만든다.
3. `projects/{slug}` 아래에 문서, 메타데이터, React 앱, E2E 시나리오를 생성한다.
4. registry 동기화와 workspace 의존성 연결 후 타입 검사, 프로덕션 빌드, 실제 브라우저 플레이테스트를 수행한다.
5. 성공·실패 단계와 산출물 경로를 `factory-report.json`에 기록한다.

## 성공 기준

- API 키 없이도 완전한 오프라인 생성 경로가 동작한다.
- `OPENAI_API_KEY`가 있으면 ChatGPT 호환 Responses API로 블루프린트를 생성할 수 있다.
- 모델 출력은 코드로 직접 실행하지 않고 스키마 검증 후 데이터로만 템플릿에 주입한다.
- 생성된 프로젝트는 launcher registry에 등록되고 정적 빌드된다.
- 자동 시나리오가 핵심 루프와 목표 완료를 실제로 수행한다.
- 실패한 단계는 비정상 종료되고 리포트에 남는다.
