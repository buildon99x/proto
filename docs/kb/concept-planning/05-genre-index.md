# 05 — 장르 인덱스 (Genre Index)

> **목적:** 골든셋(§02 성공 55 + §03 실패 22 + §06 주류 40 = **117종**)을 **장르축으로 재분류**해, *주요 장르마다 10종 이상* 확보됐는지 한눈에 검증한다. (목표: "장르별 10종 이상 + 현 시장 주류 포함.")
> **표기:** `S#` = §02 성공, `F#` = §03 실패, `M#` = §06 주류 사례 번호. 한 게임은 *주(primary) 장르*에 1회 계수, 걸치는 장르는 *교차(cross)*로 표시(계수 제외).
> **갱신일:** 2026-06-14.

## 9. 본 프로젝트 우선순위(읽기 순서)

덱빌딩 로그라이크 프로젝트이므로 **G1 → G2 → G5 → G6 → G3 → G4 → (G7–G10 주류)** 순으로 활용도가 높다. 컨셉 발굴(`docs/concepts`) 시 G1·G2에서 *차별화 어휘*를, G3·G7–G9에서 *레드플래그·리텐션 교훈*을, G10에서 *UGC·라이브옵스* 패턴(콘텐츠-as-data 전이)을 가져온다. 주류 장르(G7–G10)는 *직접 경쟁 대상이 아니라* 네트워크 효과·리텐션·운영 교훈의 원천이다(§06 횡단 교훈).
