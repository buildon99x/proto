# 02 — 골든셋: 성공 사례 34종 (역기획)

> **포맷:** 각 사례를 §01 템플릿의 핵심 칸으로 압축. **판정·지표 → 로그라인 → 판타지/미학 → X+Y 차별화 → 코어 루프 → 성공 원인·컨셉 교훈.** 수치는 신뢰도 태그([H]/[M]/[L]). 분석은 *기획적 역설계*이며 원작 소스 열람이 아니다.
> **계층:** 핵심층(1–18, 덱빌더·로그라이크·인디 — 이 프로젝트 직결, 깊게) · 확장층(19–34, 전 장르 — 보편 교훈).

---

## 핵심층 — 덱빌더 · 로그라이크 · 로그라이트 (1–12)

### 1. Slay the Spire (2019 · Mega Crit · 로그라이크 덱빌더)
- **판정:** ✅ 장르 정의작. 라이프타임 **10M+** [H], 97% 압도적 긍정.
- **로그라인:** "탑을 오르며 매 전투 보상으로 덱을 쌓아 시너지 엔진을 완성한다."
- **판타지/미학:** *전략가의 빌드 완성.* Challenge+Expression+Discovery.
- **X+Y:** 로그라이크 + **카드 덱빌딩**(장르 자체를 정의).
- **코어 루프:** 분기맵 노드 → 전투(Intent 텔레그래프, 에너지3/드로우5) → 카드/유물 보상 → 덱 정제 → 보스.
- **교훈:** **완전정보 Intent**가 "흥미로운 결정"(P9)을 매 턴 만든다. *덱 압축(deck-thinning)*이 빌드를 선명하게 — 이 프로젝트 모든 후크의 기준선. I11.

### 2. Balatro (2024 · LocalThunk(1인) · 로그라이크 덱빌더)
- **판정:** ✅ **5M+** (출시 1개월 1M) [H], TGA 베스트 인디·데뷔·모바일 석권.
- **로그라인:** "포커 족보를 덱빌딩으로 개조해 점수를 천문학적으로 폭주시킨다."
- **판타지/미학:** *룰을 부수는 도박꾼.* Discovery+Sensation(숫자 폭주).
- **X+Y:** 덱빌더 + **포커**(누구나 아는 게임필).
- **코어 루프:** 핸드 → 족보 완성 → 칩×배수 폭주 → 블라인드 돌파 → 상점 조커 구매.
- **교훈:** I1·I3·I4·I15의 교과서. *친숙한 게임필(포커) + 곱연산 폭주의 30초 가독성 + 1인 스코프.* (전체 역기획은 §01 예시.)

### 3. Inscryption (2021 · Daniel Mullins · 덱빌더+메타호러)
- **판정:** ✅ **1M+** [M], 2021 다수 GOTY 후보·수상.
- **로그라인:** "음산한 오두막에서 카드게임을 하다, 게임 자체가 탈출방·메타호러로 변형된다."
- **판타지/미학:** *규칙 너머를 의심하는 불안.* Discovery+Narrative+Sensation(공포).
- **X+Y:** 덱빌더 + **탈출방/메타호러(4번째 벽 붕괴)**.
- **코어 루프:** 카드 전투 → 방 탈출 퍼즐 → 메타 레이어 전환(룰 자체가 바뀜).
- **교훈:** **"테마=메커닉"의 절정** — 호러 톤이 룰 변형으로 *플레이된다*. 단 1회성 아크(P6) 비중이 커 리플레이는 약함 → 컨셉서에 *아크형 vs 루프형* 명시 필요.

### 4. Monster Train (2020 · Shiny Shoe · 덱빌더)
- **판정:** ✅ **1M+** [M], 압도적 긍정.
- **로그라인:** "3층 수직 전장을 방어하며 두 진영을 조합해 악마 군단을 키운다."
- **판타지/미학:** *수직 요새 사령관.* Expression(2진영 조합)+Challenge.
- **X+Y:** 덱빌더 + **타워디펜스 수직 레인 포지셔닝**.
- **코어 루프:** 적이 위층으로 침공 → 층별 유닛 배치/버프 → 콤보 폭발 → 클랜 조합 해금.
- **교훈:** I2의 증거 — *메커닉 축(수직 위치전 + 2클랜 조합)*으로 차별. "조합의 곱(클랜²)"이 리플레이를 만든다.

### 5. Hades (2020 · Supergiant · 액션 로그라이크)
- **판정:** ✅ **10M+** [H], BAFTA·휴고상, 내러티브 로그라이크 표준.
- **로그라인:** "지옥을 탈출하며 매번 죽어 돌아올 때마다 신들·가족과의 관계가 깊어진다."
- **판타지/미학:** *죽어도 따뜻한 귀환.* **Fellowship/Narrative**(희소) + Challenge.
- **X+Y:** 액션 로그라이크 + **관계/연애 서사가 진행되는 메타**.
- **코어 루프:** 던전 질주 → 신의 권능(Boon) 선택 → 사망 → 본거지서 관계·해금 진전 → 재도전.
- **교훈:** I5·I14 — **죽음을 서사로 흡수**(P10), 로그라이크 반복을 *정서 진행*으로 정당화. "반복 런의 디제시스"의 황금 표준.

### 6. Vampire Survivors (2022 · poncle · 불릿헤븐 로그라이트)
- **판정:** ✅ 메가히트(저가 $4.99, 수백만) [H], BAFTA 베스트 게임 2023.
- **로그라인:** "움직이기만 하면 자동 공격 — 화면을 뒤덮는 적을 레벨업으로 갈아버린다."
- **판타지/미학:** *약자에서 신이 되는 파워 판타지.* Sensation(물량 폭발)+Submission.
- **X+Y:** 서바이벌 + **역(逆)뱀서물 + 방치형 자동전투의 미니멀리즘**.
- **코어 루프:** 자동공격 생존 → 경험치 → 무기 진화 선택 → 화면 가득 빌드 폭주 → 30분 생존.
- **교훈:** I3·I4의 극단 — **toy-first**(움직임만으로 쾌감) + 한 장 가독성. *초저예산·초단순*이 신장르를 열었다.

### 7. Cobalt Core (2023 · Rocket Rat · 우주 함선 덱빌더)
- **판정:** ✅ **~96% 긍정** [H], 평단 호평.
- **로그라인:** "우주선을 좌우로 움직여 포격 라인을 맞추며 카드로 함선전을 벌인다."
- **판타지/미학:** *함장의 위치 싸움.* Challenge(위치 퍼즐)+Fellowship(승무원).
- **X+Y:** 덱빌더 + **함선 좌우 위치전(Into the Breach식 완전정보)**.
- **코어 루프:** 적 Intent 확인 → 함선 이동으로 포격 정렬/회피 → 카드 플레이 → 승무원 조합 해금.
- **교훈:** I2 — 같은 덱빌더라도 *위치(positioning)* 한 축으로 신선해진다. 이 프로젝트 우주/해전 컨셉의 직접 레퍼런스.

### 8. Loop Hero (2021 · Four Quarters · 카드배치 오토배틀)
- **판정:** ✅ 첫 주 50만+ [M], 컬트 히트.
- **로그라인:** "영웅은 자동으로 루프를 돌고, 너는 카드로 *세계를 깔아* 적과 보상을 배치한다."
- **판타지/미학:** *세계를 설계하는 신.* Discovery+Expression+Submission.
- **X+Y:** 오토배틀 로그라이트 + **타일 카드 배치(역할 전도: 전투는 자동, 환경을 빌드)**.
- **코어 루프:** 자동 전투 관전 → 타일 카드로 적·지형 배치(리스크↑보상↑) → 자원 → 기지 건설.
- **교훈:** I1·I11 — *제어 대상의 전도*(싸우지 않고 무대를 짠다)라는 신선한 결정구조.

### 9. Dead Cells (2018 · Motion Twin · 로그라이트 메트로배니아)
- **판정:** ✅ **10M+** [H].
- **로그라인:** "죽을 때마다 무기를 바꿔가며 변형되는 성을 베어 나가는 액션."
- **판타지/미학:** *흐르는 액션의 숙련.* Challenge(스킬)+Discovery.
- **X+Y:** 로그라이트 + **메트로배니아 + 손맛 액션**.
- **코어 루프:** 질주·전투 → 변이(빌드) 선택 → 사망 → 영구 해금 → 더 깊은 층.
- **교훈:** *스킬 기반 로그라이트*(P5 학습곡선)의 표준. 액션 손맛 = toy-first.

### 10. Risk of Rain 2 (2020 · Hopoo · 3D 협동 로그라이트)
- **판정:** ✅ **4M+** [M], 2D→3D 전환 성공.
- **로그라인:** "시간이 흐를수록 강해지는 적과의 군비경쟁 — 아이템을 무한 중첩해 폭주한다."
- **판타지/미학:** *기하급수 성장의 쾌감.* Sensation+Fellowship(협동).
- **X+Y:** 로그라이트 + **3D TPS + 시간압박 스케일링 + 협동**.
- **코어 루프:** 탐험·전투 → 아이템 스택 → 시간↑난이도↑ → 텔레포터 보스 → 다음 스테이지.
- **교훈:** I6 — 협동(Fellowship)과 *아이템 무한 중첩*이 CAR의 유능감을 폭발시킨다.

### 11. Cult of the Lamb (2022 · Massive Monster · 로그라이트+경영)
- **판정:** ✅ 첫 주 **1M** [H], 이후 수백만.
- **로그라인:** "던전을 정복하는 양(羊)이 되어, 귀환해 추종자 사교 집단을 경영한다."
- **판타지/미학:** *귀여운 광신 교주.* Expression(컬트 운영)+Discovery.
- **X+Y:** 액션 로그라이트 + **기지 건설/경영 시뮬(두 루프 교차)**.
- **코어 루프:** 던전 토벌(로그라이트) → 자원·추종자 획득 → 마을 경영·의식 → 재출정.
- **교훈:** I1 — 이질적 두 루프(액션+경영)를 *상호 보급*으로 엮어 세션 리듬을 다층화. 강한 비주얼 후크.

### 12. Darkest Dungeon (2016 · Red Hook · 파티 로그라이크 RPG)
- **판정:** ✅ 수백만 [M], 컬트 클래식.
- **로그라인:** "영웅들의 *정신*을 관리하라 — 공포·스트레스가 신체 부상보다 무섭다."
- **판타지/미학:** *소모되는 인간들의 비극.* Challenge+Narrative(고딕 호러).
- **X+Y:** 파티 RPG 로그라이크 + **스트레스/Affliction 심리 시스템**.
- **코어 루프:** 던전 탐험(횃불·스트레스 관리) → 전투 → 귀환 후 영웅 치료/해고 → 마을 강화.
- **교훈:** I5·I12 — *심리 자원*이라는 새 축이 톤(고딕 비극)을 메커닉으로 구현. 영웅이 *소모품*이라는 가혹한 정직함.

## 핵심층 — 인디 (비로그라이크, 13–18)

### 13. Stardew Valley (2016 · ConcernedApe(1인) · 농장 라이프시뮬)
- **판정:** ✅ **30M+** [H]. 1인 개발 신화.
- **로그라인:** "지친 도시인이 물려받은 농장에서 작물·관계·마을을 가꾸며 다른 삶을 산다."
- **판타지/미학:** *느린 안식과 성장.* Submission(편안함)+Expression+Fellowship.
- **X+Y:** 농사 시뮬 + **RPG/관계/탐험의 다층 루프(하베스트문 계승·확장)**.
- **코어 루프:** 하루 시간관리(농사·채광·낚시·관계) → 계절·축제 → 마을·자기 성장.
- **교훈:** I15 — *1인이 사랑으로 장르를 부활*. 다축 루프가 다양한 동기(P11)를 모두 흡수.

### 14. Hollow Knight (2017 · Team Cherry · 메트로배니아)
- **판정:** ✅ 수백만(15M+ 추정) [M], 장르 정점.
- **로그라인:** "스러진 곤충 왕국의 폐허를 탐험하며 비밀과 실력을 함께 넓혀간다."
- **판타지/미학:** *고독한 탐험의 깊이.* Discovery+Challenge(정밀 전투).
- **X+Y:** 메트로배니아 + **소울라이크 난이도 + 침묵의 분위기 서사**.
- **코어 루프:** 탐험 → 능력 해금 → 새 지역 접근 → 보스 → 지도 확장.
- **교훈:** I3(강한 아트 실루엣 가독성) + *분량/완성도*로 입소문. 톤과 아트가 곧 마케팅.

### 15. Disco Elysium (2019 · ZA/UM · 내러티브 RPG)
- **판정:** ✅ **1M+** [M], 서사 게임 다수 GOTY.
- **로그라인:** "전투 없는 형사 RPG — 너의 *스킬들이 머릿속에서 말다툼*하며 사건과 자아를 수사한다."
- **판타지/미학:** *분열된 자아의 탐구.* Narrative+Expression+Discovery.
- **X+Y:** CRPG + **전투 제거 + 내면 스킬 의인화 + 문학적 밀도**.
- **코어 루프:** 대화·조사 → 스킬 체크(주사위) → 내면 목소리 분기 → 정치/심리 선택.
- **교훈:** I9·I11 — *코어 동사를 "대화"로* 재정의. "전투를 뺀다"는 과감한 컨셉 결정이 정체성이 됐다.

### 16. Outer Wilds (2019 · Mobius · 지식기반 탐험)
- **판정:** ✅ 평단 절정, BAFTA 베스트 게임 2020 [H].
- **로그라인:** "22분마다 태양이 폭발하는 타임루프 — *오직 네가 배운 지식*만 다음 루프로 이어진다."
- **판타지/미학:** *순수한 발견의 경이.* **Discovery**(극단).
- **X+Y:** 오픈 탐험 + **타임루프 + 능력 성장이 아닌 *지식* 성장**.
- **코어 루프:** 탐사 → 단서 학습 → 루프 리셋 → *플레이어 머릿속*만 진행 → 우주 미스터리 해명.
- **교훈:** P5·P8의 정수 — 성장이 *플레이어의 이해*에 있다. 1회성 마법(스포일러 못 견딤)이라 컨셉서에 *재플레이 한계* 명시.

### 17. Lethal Company (2023 · Zeekerss(1인) · 협동 호러)
- **판정:** ✅ 2023말 폭발(스팀 1위권, 동접 24만+ [M]), 1인 개발.
- **로그라인:** "버려진 위성에서 할당량을 채우러 고철을 줍는 저예산 우주 청소부 — 괴물이 가득하다."
- **판타지/미학:** *무능한 잡부들의 공포 코미디.* Fellowship(혼돈의 협동)+Sensation.
- **X+Y:** 협동 호러 + **할당량(quota) 압박 + 보이스챗 코미디 창발**.
- **코어 루프:** 행성 착륙 → 고철 수집(괴물 회피) → 할당량 정산 → 상점 → 더 위험한 행성.
