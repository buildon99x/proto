# Spec

- Runtime: Vite React static artifact.
- Shared shell: `@prototype-lab/shared-ui` `ConceptGame`.
- Core verb: trade bottled weather.
- Win: `Profit >= 44`.
- Loss: `Reputation <= 0`.
- Required interactions: district selection, contract card play, weather stock changes, stability/reputation tradeoffs.
- UX signals: profit target, reputation danger, climate stability, rain/sun resources.
