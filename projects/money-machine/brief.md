# Money Machine Brief

## 상태

이름만 확정된 초기 스캐폴딩 단계. 아래 내용은 "머니머신"이라는 이름에서 유추한
초안 컨셉이며, 실제 구현 전에 사용자와 방향을 재확인해야 한다.

## 가정한 목표

플레이어가 자동으로 돈을 벌어들이는 "머니머신"을 클릭/구매/업그레이드로 키워
방치 중에도 수익이 쌓이는 재미를 체감하게 한다.

## 가정한 플레이어 경험

플레이어는 클릭으로 초기 자본을 모으고, 자동 수익을 내는 설비(머신)를 구매·업그레이드한다.
설비가 늘어날수록 방치 중 수익 속도가 증가하고, 누적 자본으로 더 상위 설비/배율 업그레이드를
잠금 해제한다.

## 필수 화면 (초안)

- 메인 화면 (자본 표시, 클릭 버튼, 보유 머신 목록)
- 머신 구매/업그레이드 패널
- 누적 통계 / 배율 요약

## 성공 기준 (초안)

- 첫 화면에서 설명 없이 클릭으로 자본을 얻을 수 있어야 한다.
- 머신을 구매하면 방치 중에도 자본이 자동으로 증가해야 한다.
- 업그레이드가 수익 속도에 눈에 띄게 반영되어야 한다.

- 타겟 플랫폼(모바일/데스크톱), 세션 길이, 수익화 여부 등 추가 요구사항 확인 필요.
