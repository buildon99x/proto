# Evaluation

## 상태

스펙이 확정되지 않아 아직 실질적인 평가 체크리스트는 없다. 아래는 `brief.md`의 초안
컨셉을 기준으로 한 가정 체크리스트이며, 스펙 확정 후 갱신한다.

## 가정한 체크리스트

- [ ] 첫 진입 시 클릭 버튼이 보이고 클릭 시 자본이 즉시 증가한다.
- [ ] 설비를 구매하면 방치 중에도 자본이 자동으로 증가한다.
- [ ] 업그레이드 구매 후 수익 속도 변화가 화면에 반영된다.
- [ ] `pnpm --filter money-machine dev`로 로컬 실행이 정상 동작한다.
- [ ] `pnpm --filter money-machine build`가 에러 없이 완료된다.
