# Spec

- Runtime: Vite React static artifact.
- Shared shell: `@prototype-lab/shared-ui` `ConceptGame`.
- Core verb: fold the fleet formation.
- Win: `Formation >= 45`.
- Loss: `Hull <= 0`.
- Required interactions: lane selection, fold card play, hull repair, wind management, end-turn pressure.
- UX signals: paper-sea theme, formation progress, hull danger, wind favor, run log.
