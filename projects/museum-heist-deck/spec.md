# Spec

- Runtime: Vite React static artifact.
- Shared shell: `@prototype-lab/shared-ui` `ConceptGame`.
- Core verb: read intent and steal.
- Win: `Loot >= 40`.
- Loss: `Cover <= 0`.
- Required interactions: gallery selection, stealth card play, tool/intel resource spending, curse control.
- UX signals: loot progress, cover danger, curse noise, selected gallery, log.
