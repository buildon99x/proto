# Changelog

## 0.1.0 — 프로토타입 (플레이 가능 수직 슬라이스)

도돈파치 계보의 연환 탄막 슈팅 코어를 드래곤 테마로 구현한 첫 플레이 가능 빌드.

### 추가
- **코어 엔진(Phaser):** 종스크롤 탄막 시뮬레이션 — 플레이어 이동(키보드/드래그), 샷(비늘 산탄)/레이저(용염 빔, 홀드 감속), 적탄·픽업·이펙트, 소형 피탄 판정.
- **연환(체인) 스코어링:** 연속 격파 카운터·타이머·등급(연/환/난무/용린), 곱연산 배수(난이도별 상한).
- **봄(포효)·각성(광룡화):** 화면 탄막 소거 + 무적/강화, Novice 오토봄.
- **6스테이지 캠페인 + 보스:** 데이터 주도 웨이브, 보스 3패턴(링/조준/나선), 1주차 최종보스.
- **2주차·진보스:** 노컨티뉴 클리어 시 심연 → 시원의 고룡 '아카식' → 트루 엔딩.
- **드래곤 4종:** 이그니스/글라키에스/포보스/(해금)녹스 — 샷/빔/기동/유도/근접보너스 차별.
- **메타(localStorage):** 용비늘 경제, 드래곤·난이도 해금, 드래곤별 영구 업글, 보스 도감, 연습 모드.
- **E2E 화면 플로우:** 타이틀 → 드래곤 → 난이도 → 인게임(HUD/일시정지/컨티뉴) → 리절트(정산) → 용소 허브 → 재도전.
- 런처 레지스트리 등록(`project.json`).

### 알려진 한계
- 오디오 미구현(BGM/SFX). 2주차 '심연'은 전 스테이지 리믹스 대신 진보스 단독 인카운터로 축약.
- 탄막 패턴은 프로토타입 수준(구현 시 패턴 DSL로 확장 예정).
