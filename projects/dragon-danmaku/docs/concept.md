# 컨셉 기획서 — 용린난무 龍鱗亂舞 / DRAGONPACHI

> `docs/kb/concept-planning/01-concept-template.md`의 13섹션 양식을 따른다. 카피 베이스 = **도돈파치(DoDonPachi, CAVE)**. 테마 = 드래곤·동양 신화.

---

### 0. 한 줄 식별 (Identity Line)

**용린난무(龍鱗亂舞) / DRAGONPACHI** — 종스크롤 탄막 슈팅(스코어러) · 웹(데스크톱+모바일) · 고전 STG/스코어러 코어 + 캐주얼 입문층 · 프리미엄(무 MTX).

### 1. 로그라인 (Logline)

"수호룡이 되어 적탄을 **연환(체인)**으로 엮어 점수를 폭주시키고, 마룡 군단의 화염 탄막을 비늘 산탄과 용염 빔으로 가르는 — '한 판 더'의 도박 같은 종스크롤 탄막 슈팅."

### 2. 핵심 판타지 & 타겟 감정

*나는 환생을 거듭하는 마지막 수호룡이 되어, 하늘을 메운 화염 탄막을 정밀하게 가르며 연환으로 점수를 터뜨린다.*
- **1순위 미학 = Sensation**(탄막을 헤집고 연환 배수가 폭주하는 감각적 쾌감).
- 2순위 **Challenge**(고밀도 패턴 암기·정밀 회피 숙달). 3순위 **Discovery**(숨은 비늘·드래곤 빌드·진보스 발견).

### 3. 세계관 & 톤 (World & Tone)

천공 대륙 '용들의 시대'. 역병에 물든 흑룡 군단(마룡왕)이 하늘을 침식하고, 마지막 수호룡들이 맞선다. 톤 한 단어: **"장엄한 동양 신화 + 격렬한 화염 탄막."**
- **죽음/리트라이의 디제시스적 정당화:** 수호룡은 죽으면 *환생의 알*에서 다시 태어난다(잔기=환생, 컨티뉴=용혼 소생). 2주차 '심연 회랑'은 *수호룡이 마룡왕의 근원을 추적해 시원으로 거슬러 오르는* 서사.
- 톤 레퍼런스: 동양 수묵 신화화 + 〈오쿠덴〉식 화염 이펙트 / CAVE 도돈파치 시리즈의 붉은 탄막 미학.

### 4. 코어 루프 (Core Loop)

`적 편대 등장 → 샷/레이저 선택(광역 vs 정밀) → 끊김 없이 연속 격파해 연환 유지 → 적탄을 정수 비늘로 흡수·배수 폭증 → 위기 시 봄/각성 → 미드보스·보스 격파 → 리절트 정산(점수·용비늘) → 허브 해금 → 다음 런`

- **코어 동사 = "탄막을 가르며 연환을 엮는다."** toy-first 통과: 적을 연속으로 부수며 배수 숫자가 튀는 것만으로 쾌감.
- 평균 세션: 1런(1주차) ~25–30분, 1스테이지 ~5–6분.

### 5. 핵심 차별화 & 흥미로운 결정

**우리 = 도돈파치(연환 스코어링 + 샷/레이저) + 드래곤 신화 리스킨 + 영구 성장 메타.**
- 차별축(메커닉): *연환 게이지를 유지·폭증시키는 샷/레이저 운영* + *숨은 용비늘 순서 수집*이 점수·메타 재화로 이중 환산.
- **가장 흥미로운 결정:** "지금 레이저로 보스를 빨리 녹여 안전을 살까, 샷으로 잡몹을 훑어 **연환을 더 길게 이어 배수를 살까**" — 안전 vs 점수의 매 순간 트레이드오프. 추가로 "봄을 죽음 회피로 쓸까, **노봄 보너스**로 아낄까."

### 6. 핵심 피처 적합 (Feature Fit / Scope)

- 코어 후크(탄막+연환)는 캔버스 2D 슈팅 엔진(Phaser)으로 충분히 담긴다 — 추가 엔진 비용 없음.
- 추가 비용: ① **결정적 점수 엔진**(연환 곱연산 정수 수학) ② **탄막 패턴 DSL/데이터**(패턴 튜닝·리믹스 재사용) ③ 숨은 비늘 순서 판정. 모두 데이터 주도로 분리해 정당화.
- 컷(스코프 통제): 온라인 멀티·실시간 리더보드 서버는 **MVP에서 컷**(로컬 기록·고스트로 대체), 분기 스토리·풀 컷신은 컷(간결한 인터타이틀로 대체).

### 7. 캐릭터 / 클래스 (플레이어블 드래곤)

각 드래곤이 도돈파치 A/B/C 기체처럼 *서로 다른 게임필*을 약속하고 1순위 미학(Sensation)을 분담한다. 상세 → `design/characters.md`.

1. **화룡 이그니스** — 밸런스. 넓은 비늘 산탄 + 강한 용염 빔. 입문 표준.
2. **빙룡 글라키에스** — 고화력·저속. 좁고 강한 빔, 피격 시 주변 적 둔화 유틸. 정밀형.
3. **뇌룡 포보스** — 고기동·유도 산탄·약한 빔. 회피·연환 유지 특화.
4. **(해금) 암흑룡 녹스** — 하이리스크·하이리턴. 근접 격파 시 연환 배수 가산, 잔기 보너스 축소. 스코어러 전용.

### 8. 비주얼 & 톤앤매너

- **(a) 톤앤매너 키워드:** "장엄한 수묵 신화 + 작렬하는 주홍 탄막."
- **(b) 컬러 팔레트:** 배경=먹빛 청록·군청(천공/심연)에 금빛 구름. 적탄=주홍·자홍(도돈파치 시그니처), 아군 빔=청백 코어. **강조색=금/청백**(점수·연환 배수 패널).
- **(c) 라인·셰이딩:** 수묵 텍스처 배경 + 셀셰이드 드래곤. 탄막은 외곽 글로우의 하이콘트라스트 비트맵(가독 우선).
- **(d) 실루엣·디자인 언어:** 아군 드래곤=유려한 곡선·밝은 코어, 적 마룡=각지고 부식된 흑·자색 실루엣. 적탄 코어는 항상 흰 점(피탄 판정 식별).
- **(e) UI·타이포·아이콘:** 금테 두루마리 프레임 + 묵서체 제목/도트 수치 폰트. 연환·점수·봄·각성 아이콘은 화면 가장자리에 배치(중앙 탄막 가림 방지).
- **(f) 모션·VFX:** 격파 시 정수 비늘이 빨려드는 *과장된* 흡수 연출, 연환 갱신 시 화면 가장자리 금빛 맥동, 각성 발동 시 백룡 오라 + 탄막 소거 플래시. §9 청각과 짝.
- **(g) 명도대비·가독성:** 탄막은 배경보다 항상 고명도, 적탄 흰 코어로 피탄 판정 가시화. 배경 디테일은 채도를 낮춰 탄막을 띄운다.
- **(h) 레퍼런스 + 시그니처:** 레퍼런스 = 도돈파치 대왕생 탄막 / 동양 신화 수묵화 / 〈오쿠리비〉 화염. **시그니처 비주얼 = 화면을 가득 메운 주홍 탄막 사이를 청백 빔의 용이 가르고, 가장자리에서 금빛 연환 배수가 폭주하는 한 장.**

### 9. 30초 데모 컷 & 사운드

- **데모 컷:** 잡몹 편대를 샷으로 연속 격파해 연환 배수가 ×2→×8로 튀고 → 미드보스의 화염 탄막이 화면을 메우자 빔으로 가르며 감속 회피 → 위기에서 **포효(봄)**로 화면을 소거하는 30초.
- **사운드:** 동양 타악+오케스트라 메탈의 웅장한 BGM. 시그니처 SFX = 연환 갱신 시 점층되는 종소리, 정수 비늘 흡수의 차오르는 첼레스타, 각성의 용 포효. 청각 피드백이 코어 루프(연환·흡수)와 직결.

### 10. 유사 사례 & 시장 포지셔닝

- 도돈파치/대왕생(연환·샷레이저 직계) · 동방 프로젝트(탄막 대중성) · Crimzon Clover(현대 탄막+하이퍼). 
- **차별 한 줄:** 정통 도돈파치 손맛을 **웹에서 즉시 플레이**, 거기에 *드래곤 판타지 + 영구 성장 메타*로 입문 장벽을 낮춘다(고전 STG는 보통 메타 성장이 빈약).
- 타이밍: 웹 기반 탄막 STG는 희소 → 접근성(브라우저·모바일)과 무 MTX로 빈 니치.

### 11. MDA 분석

- **M(규칙):** 샷/레이저 이원 무기, 연환 곱연산 점수, 봄/각성 자원, 숨은 비늘 순서 수집, 2주차.
- **D(창발):** "연환을 어디까지 욕심내다 끊길 위험을 감수할까", 샷↔레이저 전환 타이밍, 봄 보존 vs 생존, 드래곤별 점수 빌드.
- **A(미학):** Sensation + Challenge + Discovery.
- **장점:** 검증된 도돈파치 코어 + 발견형 메타로 캐주얼까지 흡수. **가장 큰 약점:** 탄막 진입 장벽 → Novice·오토봄·연습 모드·영구 업글로 완충(단, 메타가 실력을 대체하지 않게 절제).

### 12. 메타·리텐션 & 수익

- **메타 진행:** 용비늘로 드래곤·영구 업글·난이도·연습·도감·스킨 해금(Discovery 영속화). 상세 → `design/meta-progression.md`.
- **CAR 3줄:** *유능감* = 점수 신기록·노컨티뉴·2주차 돌파. *자율성* = 드래곤 4종×샷/레이저×업글 빌드 자유, 난이도 선택. *관계성* = 로컬 기록 갱신·리플레이/고스트 공유(비대칭, 약).
- **수익·윤리선:** 프리미엄 1회 구매, **무 MTX**. 가변보상 = **발견형(숨은 비늘·진보스)**, 착취형 가챠 없음.

### 13. 사전 포스트모템 & 가드레일

1. *탄막 입문 장벽으로 캐주얼 이탈* → Novice/오토봄·연습 모드·점진 영구 업글로 온보딩, 단 업글 폭 절제로 코어 실력 가치 보존.
2. *메타 성장이 P2W처럼 점수 인플레를 일으켜 스코어러 신뢰 상실* → 영구 업글은 생존 편의 위주, **랭킹/기록은 업글 영향 최소화·구분 집계**.
3. *카피 베이스라 'me-too'로 보임* → 드래곤 테마·연환 이중 환산·웹 접근성·메타 볼륨으로 차별, 시그니처 비주얼(§8h)로 정체성 확보.

> **검수 결론:** 위 13칸만 읽으면 도돈파치를 모르는 사람도 *어떤 게임인지·왜 재밌는지·3–4시간을 어떻게 채우는지*가 그려진다.
