# 메타 성장 · 경제 · 난이도 · 플레이타임 예산

> 3–4시간+ 플레이타임의 근거. 재화 = **용비늘(Scales)**. 무 MTX, 발견형 보상, 코어 실력 가치 보존을 원칙으로 한다.

## 1. 용비늘(Scales) 획득

런 1회당 정산식: `획득 용비늘 = floor(점수/기준치) + 숨은비늘 수집수 × 보너스 + 클리어 보너스(스테이지수·노미스·노봄·노컨티뉴)`.
- 점수·수집·생존이 모두 재화로 환산 → 잘할수록·멀리 갈수록 더 빠른 해금(숙련 보상).

## 2. 허브 '용소(龍巢)' 해금/업글 트리 (전 항목)

### A. 드래곤 해금
| 항목 | 비용(용비늘) | 추가 조건 |
|---|---|---|
| 화룡 이그니스 | 0 (기본) | — |
| 빙룡 글라키에스 | 300 | — |
| 뇌룡 포보스 | 300 | — |
| 암흑룡 녹스 | 1200 | 1주차 클리어 1회 |

### B. 드래곤별 영구 업글 (각 드래곤 개별, 폭 절제)
| 업글 | 효과 | 레벨 | 레벨당 비용 |
|---|---|---|---|
| 시작 잔기 | +1 (최대 +2) | 2 | 250 / 600 |
| 봄 소지수 | +1 (최대 +2) | 2 | 200 / 500 |
| 각성 효율 | 충전속도 +10%/단계 (최대 +30%) | 3 | 150 / 300 / 500 |
| 옵션(비룡) | 보조 화력 +1 (최대 +2) | 2 | 400 / 900 |
| 정수 흡수 | 흡수 점수·타이머 연장 +10%/단계 | 2 | 200 / 450 |

> 가드레일: 화력 직접 배율 업글은 제외. 잔기/봄/각성 등 **생존·편의 위주**로 코어 난이도 가치 보존. 랭킹은 업글 적용/순수 구분 집계.

### C. 난이도 해금
| 난이도 | 해금 조건 |
|---|---|
| Novice (오토봄) | 기본 |
| Original | 기본 |
| Maniac (고밀도) | Original 1주차 클리어 |
| Ultra / 심연(2주차) | Maniac 노컨티뉴 클리어 |

### D. 편의·콘텐츠 해금
| 항목 | 비용/조건 |
|---|---|
| 연습 모드(스테이지 셀렉트) | 해당 스테이지 도달 시 자동 |
| 보스 러시 | 1주차 클리어 |
| 보스 도감 | 보스 격파 시 자동 등록 |
| 뮤직룸 | 스테이지 BGM 청취 시 등록 |
| 스킨(드래곤 컬러) | 도전과제 보상(무 MTX) |

## 3. 난이도 스케일링 요약

- 탄 밀도·탄속·패턴 단계·연환 배수 상한이 난이도별로 스케일(`mechanics.md §3`).
- Novice: 오토봄 + 저밀도 + 잔기 여유 → 온보딩. Ultra/심연: 고밀도 + 진보스 → 코어 엔드게임.

## 4. 플레이타임 예산 (3–4시간+ 산술)

| 활동 | 예상 시간 |
|---|---|
| Novice 첫 1주차 클리어(온보딩) | ~30–40분 |
| Original 1주차 노컨티뉴 숙련 | ~60–90분 |
| 드래곤 3–4종 × 샷/레이저 성향 시도 | ~60분 |
| Maniac 도전 + 2주차(심연) 진입 | ~45–90분 |
| 진 최종보스 '아카식' 공략 | ~20–40분 |
| 보스 러시 · 스코어 어택 · 도전과제 | 오픈엔드(+α) |
| **누적(코어 경로)** | **약 3.5–5시간, 리플레이 주도** |

> 핵심: 고전 STG의 짧은 런(1회 25–30분)을 **메타 해금·난이도 계단·드래곤 다양성·2주차/진보스**로 곱해 3–4시간+를 자연스럽게 채운다. 단일 클리어가 아니라 **숙련·기록 갱신**이 주 동력.
