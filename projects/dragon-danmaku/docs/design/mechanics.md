# 메커닉 설계 — 무기 · 연환 · 스코어링 · 봄 · 각성

> 도돈파치 코어 메커닉을 1:1 계승하고 수치를 본작 기준으로 설계한다. 수치는 초기 튜닝 기준값이며 데이터(DSL)로 분리해 조정한다.

## 1. 무기 — 샷 / 레이저 (이원 무기)

| 항목 | 비늘 산탄(샷, 탭/연사) | 집중 용염 빔(레이저, 홀드) |
|---|---|---|
| 입력 | 탭 연타 또는 자동 연사 토글 | 버튼 홀드 |
| 탄도 | 전방 부채꼴 확산(레벨업 시 각 증가) | 직선 관통 집중 빔 |
| 위력 | 발당 낮음, 광역 DPS | 단일 표적 고 DPS |
| 이동속도 | 100% | **약 45%로 감속(정밀 회피 모드)** |
| 역할 | 잡몹 광역 처리 → **연환 유지** | 보스·경장갑 고속 격파, 정밀 회피 |

- **파워업:** 스테이지 드롭 'P' 비늘로 무기 레벨 1→MAX. 풀파워 후 'P'는 점수 보너스로 환산.
- **옵션(비룡):** 드래곤을 따라다니는 보조 화력. 메타 업글로 최대 +2기.

## 2. 연환(連環) 스코어링 — 코어

도돈파치 체인의 직계. **격파 간격 타이머**가 핵심.

- 적 격파 시 **연환 카운터 +1**. 카운터가 화면 가장자리 게이지에 표시.
- 다음 격파가 **연환 타이머(기본 약 1.0초, 적 종류별 가감)** 내에 들어오면 카운터 유지·증가. 초과하면 **카운터 0으로 리셋**.
- 점수 배수는 카운터에 연동(아래 §3).
- **타이머 연장 요소:** 레이저 격파는 타이머를 약간 더 줌(보스전 체인 유지), 옵션/근접 격파 보너스(드래곤별).

### 연환 등급(시각·청각 피드백)
| 카운터 구간 | 등급 | 피드백 |
|---|---|---|
| 1–49 | 연(緣) | 가장자리 약한 금빛 |
| 50–149 | 환(環) | 금빛 맥동 + 종소리 1음 |
| 150–399 | 난무(亂舞) | 강한 맥동 + 종소리 점층 |
| 400+ | 용린(龍鱗) | 화면 테 전체 금빛 + 코러스 |

## 3. 스코어링 공식

격파 점수 = `기본점수 × 연환배수 × 각성배수 × 난이도계수`

- **연환배수:** `1 + floor(연환카운터 / 10) × 0.1` (예: 카운터 100 → ×2.0, 400 → ×5.0). 상한은 난이도별(Original ×8, Maniac ×12, Ultra 무제한 근사).
- **각성배수:** 각성(광룡화) 발동 중 ×2.0.
- **정수 비늘 흡수:** 적 격파/각성 시 적탄이 정수 비늘로 전환 → 흡수 시 개당 소량 점수 + **연환 타이머 미세 연장**.
- **숨은 용비늘 보너스(§5):** 순서대로 수집 시 누적 보너스(1→2→…→N), 중간 실패 시 누적 리셋(원작 벌 계승).
- **스테이지 클리어 보너스:** 노미스·노봄·풀파워·숨은 비늘 풀수집 각각 가산.
- 결정성: 모든 곱연산은 정수/고정소수로 처리해 리플레이·기록 재현 보장.

## 4. 봄 — 포효(절멸의 숨결)

- 발동: 화면 전체 적탄 소거 + 약 2초 무적 + 광역 대미지.
- 재고: 스테이지 시작 시 N개(기본 2, 메타 업글로 +). 'B' 아이템으로 보충.
- **노봄 보너스:** 스테이지를 봄 미사용 클리어 시 큰 점수 보너스 → 봄 보존 트레이드오프.
- **Novice 오토봄:** 피격 순간 봄 재고가 있으면 자동 발동(잔기 보존, 점수 보너스는 축소).

## 5. 숨은 용비늘 / 용알 수집

- 스테이지 특정 지형·오브젝트를 파괴하면 숨은 용비늘(번호 부여)이 출현.
- **정해진 순서(1→2→…)대로** 먹으면 누적 보너스가 기하적으로 증가. 순서 어기면 누적 리셋(점수만, 회수 자체는 메타 재화로 인정).
- 풀수집 시 스테이지 보너스 + 메타 재화(용비늘) 대량 획득 → 1주차 반복의 숙련 동기.

## 6. 각성 — 광룡화 (하이퍼)

- 게이지: 격파·정수 흡수로 충전. 만충 시 수동 발동.
- 효과(약 8–12초): 화력 강화(샷 탄수↑/빔 폭↑), **발동 순간 화면 탄막 소거**, 점수 ×2.0(각성배수), 피격 판정 변화 없음(난이도 유지).
- 트레이드오프: 보스 직전에 아껴 폭딜할지, 잡몹 구간에 터뜨려 연환·점수를 불릴지.

## 7. 피격 · 잔기 · 컨티뉴

- 히트박스: 드래곤 중심 소형 점(도돈파치식). 그레이즈(아슬 회피)는 각성 게이지 미세 충전.
- 피격 → 잔기 -1, 짧은 무적 리스폰, 무기 레벨 일부 하락.
- 잔기 0 → **컨티뉴 화면**(카운트다운). 컨티뉴 시 점수 일부 리셋·노컨티뉴 자격 상실, 거부 시 게임오버→리절트.
- **2주차(심연) 진입 조건:** 1주차를 노컨티뉴(+난이도별 추가 조건: 일정 숨은비늘 수집 또는 점수 임계)로 클리어. 미달 시 1주차 엔딩 후 리절트.
