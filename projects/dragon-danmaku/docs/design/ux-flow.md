# UX 전체 플로우 — E2E 화면 · 상태 · 전이

> 게임을 **부팅부터 재도전까지** 빈틈없이 정의한다. 모든 화면은 진입·이탈 경로를 가지며 막다른 화면이 없다. (eval.md A 체크리스트 대응)

## 1. 전체 플로우 다이어그램

```
[부팅/로고]
   ↓
[타이틀] ──→ [옵션] ──┐
   │  ├──→ [용소 허브(메타)] ──→ [도감/뮤직룸/스킨]
   │  ├──→ [연습 모드] ──→ [스테이지 셀렉트 / 보스 러시] ──→ (인게임)
   │  └──→ [게임 시작]
   ↓
[드래곤 선택] ←─(뒤로)─ [타이틀]
   ↓
[난이도 선택] ←─(뒤로)─ [드래곤 선택]
   ↓
[인게임 플레이] ⇄ [일시정지]
   │   ├─(피격·잔기↓)
   │   ├─(잔기 0)──→ [컨티뉴?] ─Yes→ (인게임 재개)
   │   │                         └No→ [게임오버]
   │   ├─(스테이지 클리어)──→ 다음 스테이지 … → [1주차 엔딩]
   │   └─(2주차 자격 충족)──→ [심연 회랑] ──→ [진보스] ──→ [트루 엔딩]
   ↓
[리절트(점수·용비늘 정산)]
   ├──→ [용소 허브] (해금·업글)
   ├──→ [재도전] ──→ [드래곤 선택]
   └──→ [타이틀]
```

## 2. 화면별 정의

### 부팅 / 로고
- 로고 → 자동으로 타이틀. (이탈: 자동 전이)

### 타이틀
- 구성: 로고, 메뉴(게임 시작 / 연습 모드 / 용소 허브 / 옵션 / 종료), 최고 기록 표시.
- 전이: 각 메뉴 → 해당 화면. (이탈: 종료 또는 하위 화면)

### 드래곤 선택
- 구성: 해금된 드래곤 4종 카드(미해금은 자물쇠+조건 표시), 스타일 토글(샷 편향/레이저 편향), 성능 요약.
- 전이: 선택 → 난이도 선택. **뒤로 → 타이틀.**

### 난이도 선택
- 구성: Novice/Original/Maniac/(Ultra·심연=조건부) + 각 설명, 시작 잔기·봄 표시(메타 업글 반영).
- 전이: 확정 → 인게임. **뒤로 → 드래곤 선택.**

### 인게임 플레이 (HUD)
- HUD(가장자리 배치, 중앙 탄막 비가림): 점수, 최고기록, **연환 카운터/등급 게이지**, 잔기, 봄 재고, **각성 게이지**, 무기 레벨, 현재 스테이지.
- 입력: 이동 / 샷 / 레이저(홀드 감속) / 봄 / 각성 / 일시정지.
- 전이: 일시정지(⇄), 스테이지 클리어→다음, 잔기 0→컨티뉴, 자격 충족→심연/진보스.

### 일시정지
- 구성: 계속 / 옵션(볼륨·키) / 포기(리절트로). (이탈: 계속 또는 포기)

### 컨티뉴?
- 트리거: 잔기 0. 카운트다운 + 잔여 컨티뉴 횟수.
- 전이: Yes → 인게임 재개(점수 일부 리셋·**노컨티뉴 자격 상실**). No/타임아웃 → 게임오버.

### 게임오버
- 구성: 사망 스테이지·최종 점수. 전이: → 리절트.

### 1주차 엔딩 / 트루 엔딩
- 1주차 엔딩: 마룡왕 격파(2주차 자격 미달 시). 트루 엔딩: 진보스 '아카식' 격파.
- 전이: → 리절트.

### 리절트 (점수·용비늘 정산)
- 구성: 최종 점수, 연환 최고, 숨은 비늘 수집수, 노미스/노봄/노컨티뉴 보너스, **획득 용비늘**, 신기록 여부.
- 전이: **용소 허브 / 재도전(→드래곤 선택) / 타이틀.** (막다른 없음)

### 용소 허브 (메타)
- 구성: 보유 용비늘, 드래곤 해금, 드래곤별 영구 업글, 난이도 해금, 연습·보스러시 개방, 도감/뮤직룸/스킨 진입.
- 전이: 각 하위 화면, **뒤로 → 타이틀(또는 직전 리절트).**

### 도감 / 뮤직룸 / 스킨
- 도감: 격파한 보스·적 정보. 뮤직룸: 청취한 BGM. 스킨: 도전과제 보상 컬러.
- 전이: **뒤로 → 용소 허브.**

### 연습 모드
- 구성: 스테이지 셀렉트(도달한 스테이지) / 보스 러시(1주차 클리어 후). 기록은 별도 집계.
- 전이: 선택 → 인게임(연습), **뒤로 → 타이틀.**

### 옵션
- 구성: 볼륨(BGM/SFX), 키/패드 리매핑, 자동 연사 토글, 모바일 컨트롤 배치, 화면 흔들림·접근성.
- 전이: **뒤로 → 호출 화면(타이틀/일시정지).**

## 3. 엣지·실패 경로 처리 (eval.md D 대응)

| 상황 | 처리 |
|---|---|
| 잔기 0 | 컨티뉴 화면 → Yes 재개 / No 게임오버 |
| 봄 0에서 위기 | 봄 비활성(아이콘 0), Novice 오토봄도 재고 없으면 미발동 |
| 연환 끊김(타이머 초과) | 카운터 0 리셋, 등급 강등 피드백, 점수 배수 하락 |
| 숨은 비늘 순서 실패 | 누적 점수 보너스만 리셋, 회수분은 메타 재화로 인정 |
| 컨티뉴 사용 | 점수 일부 리셋·노컨티뉴/2주차 자격 상실, 플레이 계속 |
| 노컨티뉴 클리어 | 2주차(심연) 자격 부여 + 자격 충족 시 자동 진입 |
| 2주차 자격 미달 | 1주차 엔딩 → 리절트(정상 종료) |
| 모바일 이탈/백그라운드 | 자동 일시정지 |

## 4. 첫 세션 온보딩(무튜토리얼 지향)

- 드래곤 선택에 "첫 플레이 권장: 이그니스 + Novice" 힌트.
- 1스테이지 도입부에 비침습적 조작 안내(샷/레이저/봄) 오버레이.
- Novice 오토봄으로 초보 사망 완충 → 리절트에서 소액 용비늘 → 첫 해금 경험으로 메타 루프 진입.
