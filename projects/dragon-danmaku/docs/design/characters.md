# 플레이어블 드래곤 설계

> 도돈파치 A/B/C 기체 개념 계승 — 각 드래곤 × 샷/레이저 성향으로 서로 다른 게임필을 제공한다. 리플레이 동력의 핵심.

## 공통 구조

- 각 드래곤은 **샷 타입**(확산 비늘 산탄)과 **레이저 타입**(집중 용염 빔)을 모두 보유, 성향 수치가 다름.
- 선택 화면에서 드래곤 + (스타일: 샷 편향 / 레이저 편향) 선택 → 도돈파치의 기체×샷타입 조합과 동치.

## 드래곤 4종

### 1. 화룡 이그니스 (Ignis) — 밸런스 / 입문 표준
- 샷: 넓은 부채꼴 산탄(연환 유지 우수). 빔: 표준 폭·위력.
- 이동속도 중간, 봄 위력 표준. **첫 플레이 권장.**
- 판타지: 정공법의 수호룡.

### 2. 빙룡 글라키에스 (Glacies) — 고화력 / 저속 / 정밀
- 샷: 좁고 강한 집중 산탄. 빔: 최고 위력·관통.
- 이동 느림(정밀 회피 유리), 피격 시 주변 적 **둔화** 유틸 1회.
- 판타지: 한 발 한 발 얼려 부수는 냉혈 포식자. 보스 버스트·정밀 스코어러용.

### 3. 뇌룡 포보스 (Phobos) — 고기동 / 유도 / 연환 유지
- 샷: **유도 비늘**(자동 표적, 연환 유지 최강). 빔: 약함·짧음.
- 이동 빠름. 잡몹 구간 체인 유지·회피 특화, 보스 화력은 약점.
- 판타지: 번개처럼 누비는 기동형.

### 4. (해금) 암흑룡 녹스 (Nox) — 하이리스크 / 하이리턴 / 스코어러
- 샷: 표준이나 **근접 격파 시 연환 배수 가산** 보너스. 빔: 강하나 사거리 짧음.
- 잔기·봄 시작 재고 축소(고위험), 점수 상한 해방.
- 해금: 용비늘 비용(meta-progression 표) + 1주차 클리어 조건. 스코어 어택 전용.
- 판타지: 금기의 힘을 다루는 타락 직전의 용.

## 밸런스 가드레일

- 4종 모두 1주차 클리어 가능(접근성). 차이는 **점수 상한과 운영 난이도**(스코어링 다양성), 단순 강약이 아님.
- 메타 영구 업글은 드래곤별로 적용되나 폭을 절제(코어 실력 가치 보존 — concept §13 가드레일).
