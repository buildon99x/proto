# Brief

## Game

알톡! 드래곤 부화장은 짧은 세션 안에서 반복 클릭, 성장, 랜덤 보상, 수집, 자동화를 모두 체감하는 2D 클릭커 웹 프로토타입이다.

## Target Player

- 클릭커와 수집형 캐주얼 게임의 즉각적인 보상을 좋아하는 플레이어
- 복잡한 튜토리얼 없이 중앙의 알을 누르며 성장 흐름을 이해하고 싶은 플레이어
- 드래곤 도감 수집과 희귀 보상 연출을 기대하는 플레이어

## First Useful Workflow

1. 중앙의 알을 클릭한다.
2. 부화 게이지와 온기가 오른다.
3. 균열 이벤트가 뜨면 즉시 클릭해 Perfect Hatch 보너스를 받는다.
4. 알이 부화하면 랜덤 드래곤을 얻고 도감에 등록한다.
5. 드래곤 자동 생산으로 온기, 마나, 골드를 모은다.
6. 업그레이드와 상위 알 해금으로 더 높은 등급 보상을 노린다.
