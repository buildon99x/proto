# Spec

## MVP Scope

- 메인 부화 화면: Phaser 캔버스 기반 중앙 알, 클릭 애니메이션, 부화 게이지, Perfect Hatch 균열 연출
- 드래곤 도감: 12종 드래곤, 획득 수량, 신규 등록 강조, 미발견 상태
- 업그레이드 패널: 6종 업그레이드, 레벨, 비용, 구매 가능/불가 상태
- 알 선택/구매 패널: 기본 알, 희귀 알, 고대 알 3종과 비용/해금 상태
- 자원: 온기, 마나, 골드
- 등급: Common, Rare, Epic, Legendary

## Core Rules

- 클릭 시 활성 알의 부화 게이지가 클릭 파워만큼 증가한다.
- 클릭 시 활성 알 설정에 따라 온기/마나/골드를 얻는다.
- 일정 확률로 빛나는 균열 이벤트가 발생한다.
- 균열 이벤트 중 다음 클릭은 Perfect Hatch로 처리되어 게이지 상승량이 증가한다.
- 게이지가 100%에 도달하면 알이 부화하고 등급 가중치에 따라 드래곤 1종이 랜덤 지급된다.
- 획득한 드래곤은 도감에 등록되며 초당 자동 생산량에 더해진다.
- 업그레이드는 자원을 소비하고 클릭 파워, 자동 생산량, 희귀 확률, 균열 보너스, 보상량을 강화한다.
- 상위 알은 자원으로 구매 후 선택 가능하며, 선택 시 게이지는 새 알 기준으로 초기화된다.

## Implementation

- Vite + React + TypeScript 앱 안에서 Phaser가 중앙 알 무대와 클릭 피드백을 렌더링한다.
- 게임 데이터는 `src/data.ts`, 규칙과 상태 전이는 `src/gameLogic.ts`, Phaser 브리지는 `src/EggStage.tsx`에 둔다.
- 텍스트가 많은 도감, 업그레이드, 알 패널은 DOM UI로 유지한다.
