# 대장장이 클릭커 Brief

## 목표

짧은 세션 안에 반복 클릭, 랜덤 무기 제작, 선택 보상, 대장간 성장, 자동화가 연결되는 클릭커 장르의 핵심 재미를 체감하게 한다.

## 플레이어 경험

플레이어는 귀여운 2D 판타지 대장간의 대장장이가 되어 중앙 모루를 클릭한다. 제작 게이지가 100%에 도달하면 무기가 완성되고, 판매/분해/강화/보관 중 하나를 즉시 선택한다. 선택 결과는 골드, 재료, 강화 수치, 도감 기록으로 바로 피드백된다.

## 필수 화면

- 메인 제작 화면
- 무기 완성 결과 모달
- 업그레이드 패널
- 재료 인벤토리
- 무기 보관함/도감

## 성공 기준

- 첫 화면에서 별도 설명 없이 중앙 모루 클릭이 가능해야 한다.
- 첫 무기 완성 후 네 가지 선택지가 모두 의미 있게 동작해야 한다.
- 골드와 재료가 업그레이드, 강화, 보관 루프에 연결되어야 한다.
- 자동 제작을 구매하면 방치 중에도 제작 진행도가 증가해야 한다.
