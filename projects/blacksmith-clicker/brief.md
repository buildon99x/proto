# 대장장이 클릭커 Brief

## Goal

짧은 세션 안에서 클릭커 장르의 핵심 재미를 체감할 수 있는 2D 판타지 대장간 웹 프로토타입을 만든다. 플레이어는 모루 위 무기를 직접 클릭해 제작 게이지를 채우고, 완성된 무기를 판매, 분해, 강화, 보관 중 하나로 처리하며 대장간을 성장시킨다.

## Player Fantasy

플레이어는 귀엽고 바쁜 대장장이이다. 망치질마다 불꽃이 튀고 금속이 달아오르며, 희귀한 무기가 완성될 때 강한 빛과 흔들림으로 보상이 느껴져야 한다.

## Core Loop

1. 중앙 작업대의 무기/모루를 클릭한다.
2. 제작 게이지가 상승한다.
3. 게이지가 100%에 도달하면 무기가 완성된다.
4. 종류, 등급, 품질, 속성이 랜덤으로 정해진다.
5. 완성 무기를 판매, 분해, 강화, 보관 중 하나로 처리한다.
6. 골드와 재료를 사용해 업그레이드와 강화를 진행한다.
7. 업그레이드로 더 빠르고 희귀한 제작을 반복한다.

## Required Screens

- 메인 제작 화면
- 무기 완성 결과 모달
- 업그레이드 패널
- 재료 인벤토리
- 무기 보관함/도감

## Experience Target

첫 30초 안에 최소 1개의 무기를 만들고 보상 선택을 경험한다. 2분 안에 판매 또는 분해를 통해 업그레이드를 구매하고 자동 제작이 성장하는 감각을 확인할 수 있어야 한다.
