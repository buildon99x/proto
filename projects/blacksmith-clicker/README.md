# 대장장이 클릭커

2D fantasy blacksmith clicker prototype. Click the forge to craft random weapons, then sell, salvage, enhance, or store them to grow the forge.

## Commands

- `corepack.cmd pnpm --filter blacksmith-clicker dev`
- `corepack.cmd pnpm --filter blacksmith-clicker test`
- `corepack.cmd pnpm --filter blacksmith-clicker build`
