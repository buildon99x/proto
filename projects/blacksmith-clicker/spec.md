# 대장장이 클릭커 Spec

## 런타임

- Vite React single page app
- DOM 기반 HUD와 CSS 기반 2D 제작 장면
- 정적 빌드 결과는 launcher의 `/runs/blacksmith-clicker/index.html`에서 iframe으로 실행

## 핵심 상태

- `gold`: 판매와 초기 보유 자금
- `materials`: 철광석, 화염석, 마력 결정, 고대 파편
- `levels`: 망치, 제작대, 화로, 직원, 상인, 분해, 강화 업그레이드 레벨
- `progress`: 현재 제작 진행도
- `completedWeapon`: 결과 모달에서 처분 대기 중인 무기
- `storedWeapons`: 도감/보관함에 저장한 무기 목록
- `logs`: 최근 제작과 선택 결과

## 제작

- 중앙 모루 클릭 시 `clickPower = 7 + 망치 레벨 * 2.4`만큼 진행된다.
- 제작 목표치는 `max(68, 100 - 제작대 레벨 * 5)`다.
- 직원 레벨이 1 이상이면 초당 `직원 레벨 * 1.15`만큼 자동 진행된다.
- 제작 완료 시 진행도는 0으로 초기화되고 결과 모달이 열린다.

## 무기 생성

- 종류: 검, 도끼, 창, 단검, 활, 지팡이, 망치, 방패
- 등급: Common, Rare, Epic, Legendary
- 품질: D, C, B, A, S
- 속성: 무속성, 불꽃, 얼음, 번개, 독, 신성
- 강화 레벨: +0부터 시작
- 화로 레벨은 Rare/Epic/Legendary 가중치를 올린다.
- 제작대 레벨은 상위 품질 판정에 보정치를 준다.

## 결과 선택

- 판매: 무기 가치, 상인 계약, 강화 레벨을 반영해 골드를 지급한다.
- 분해: 등급/속성 기반 재료를 지급하고 분해 기술 레벨로 획득량을 증폭한다.
- 강화: 골드와 재료를 소비해 강화 레벨과 가치를 올린다. 강화 기술은 골드 비용을 낮춘다.
- 보관: 무기를 보관함에 등록하고 도감 통계를 갱신한다.

## 피드백

- 클릭 시 망치 애니메이션과 불꽃 파티클을 보여준다.
- 제작 게이지가 오를수록 금속 색이 달아오른다.
- 완성 시 화면 밝기 펄스와 결과 모달이 등장한다.
- Rare 이상 결과는 등급 색상과 떠오르는 텍스트로 강조한다.
- Legendary 결과는 모달 흔들림과 강한 빛 효과를 사용한다.
