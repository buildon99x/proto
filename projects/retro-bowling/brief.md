# Brief — Retro Bowling

## 대상 사용자
- 짧은 시간에 즐기는 캐주얼 아케이드 게임을 찾는 웹 방문자.
- 도트/픽셀 그래픽과 8비트 감성을 좋아하는 레트로 게임 팬.
- 데스크톱(키보드)과 모바일(터치) 모두 지원.

## 문제 / 동기
- 브라우저에서 설치 없이 바로 즐길 수 있는, 규칙이 명확하고 손맛이 좋은 미니 게임이 필요하다.
- 볼링은 규칙(10프레임, 스트라이크/스페어)이 보편적으로 익숙해 학습 비용이 낮고,
  파워·스핀·조준의 3단계 입력이 아케이드 조작감과 잘 맞는다.

## 첫 유용한 경험(First useful workflow)
1. 타이틀에서 SPACE/탭으로 시작.
2. `< >` 로 좌우 조준 → SPACE 로 확정.
3. 파워 미터가 왕복할 때 SPACE 로 파워 고정.
4. 스핀(훅) 미터가 왕복할 때 SPACE 로 스핀 고정과 동시에 투구.
5. 핀이 물리로 쓰러지고, 상단 스코어시트에 표준 볼링 점수가 실시간 집계된다.
6. 10프레임을 마치면 최종 점수·랭크·하이스코어(로컬 저장)를 확인하고 재도전.

## 핵심 가치
- **도트 그래픽**: 낮은 내부 해상도를 nearest-neighbor 로 확대, 자체 5x7 비트맵 폰트, 제한된 팔레트, CRT 스캔라인.
- **손맛**: 오실레이팅 파워/스핀 미터 + 실시간 핀 물리(연쇄 반응, 스플릿, 포켓).
- **가벼움**: 외부 에셋/네트워크 없이 정적 산출물로 동작, 즉시 로드.
