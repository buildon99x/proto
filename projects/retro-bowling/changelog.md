# Changelog

## 0.1.0

- 도트 그래픽 레트로 볼링 게임 초기 구현.
- 원근 레인 렌더, 실시간 2D 핀/볼 물리(연쇄 반응·스플릿·포켓·거터).
- 파워/스핀(훅)/조준 3단계 아케이드 입력, 키보드 + 터치 지원.
- 표준 10프레임 스코어링(스트라이크/스페어/10프레임 보너스구) + 실시간 스코어보드.
- 자체 5x7 비트맵 픽셀 폰트, 제한 팔레트, CRT 스캔라인 연출.
- WebAudio 칩튠 사운드(투구/핀/스트라이크/스페어/거터/게임오버), 음소거 토글.
- 하이스코어 로컬 저장, 랭크(ROOKIE~PERFECT), 게임오버 미니 스코어시트.
- 헤드리스 플레이테스트 시나리오(tests/e2e/scenario.mjs) 추가.

## 0.2.0 — 쉬운 조작 / 어려운 완전정복

- **포켓 강제 물리 튜닝**: 정중앙 직구는 스트라이크가 나지 않고(헤드리스 스트라이크맵
  검증), 헤드핀 우측 + 좌향 훅으로 1-3 포켓에 각을 넣어야 스트라이크가 응집.
  볼 질량을 낮춰 정면 충돌 시 감속·편향되도록 함(실제 볼링의 캐리 감).
- **스윗스팟 미터**: 파워/스핀 미터에 GOOD(초록)·PERFECT(노랑) 존 표시. 오실레이팅
  마커를 존에 멈추면 기대 핀 수·스트라이크 확률↑(카오스라 보장은 아님 — 공정).
  PERFECT/GOOD 락 피드백 표시.
- **라이브 궤도 프리뷰**: 핀 무시 예상 경로를 실시간 표시. 스핀 단계에선 오실레이터
  값에 따라 훅 아크가 실시간으로 휘어져 조준을 돕는다.
- **추천 조준 마커(POCKET)**: 어디를 겨냥할지 레인에 시안 쐐기로 안내.
- 물리 상수를 `TUNE` 객체로 분리(시뮬·프리뷰·검증 하니스 공유), `predictPath` 추가.
- 밸런스 검증: 랜덤 플레이 ~55점 vs 포켓+스윗스팟 플레이 ~128점(퍼펙트 300은 여전히 극악).

## 0.3.0 — 난이도 램프 & 스플릿/스페어 심화

- **난이도 램프**: 프레임이 진행될수록 파워/스핀 미터가 빨라짐(1.0→1.6배, `METER_RAMP`).
  후반 스트라이크 스트링일수록 타이밍이 극악. 하단에 `SPD X1.4` 등으로 표시.
- **스플릿 감지**: 첫 공 후 헤드핀이 없고 잔핀이 좌우로 벌어지면 "SPLIT!" 배너 + 조준
  마커가 붉게. 7-10 등 스플릿은 한 공으로 다 못 잡음(전환 난이도↑).
- **적응형 조준 마커**: 스페어(부분 랙)에선 마커가 남은 핀 중심을 겨냥(POCKET→SPARE/
  SPLIT 라벨). 스페어는 겨냥이 쉬워지되(접근성), 스플릿은 여전히 어렵다.
- 밸런스: 적응형 스페어 조준으로 자동 플레이 ~145점. 스페어만으로는 상한이 있어
  200점 이상은 여전히 스트라이크(포켓) 정복이 필요 — 천장 유지.
