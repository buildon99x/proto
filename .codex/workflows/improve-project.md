# Improve Project Workflow

1. Read the project's `brief.md`, `spec.md`, `eval.md`, `notes/decisions.md`, and `prompts/improvement-log.md`.
2. Keep all project-specific changes inside `projects/{slug}`.
3. Update docs when behavior changes.
4. Run the project-level checks.
5. Run `pnpm sync:registry` when metadata changes.
6. Run `pnpm build:vercel` before release.
